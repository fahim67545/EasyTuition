# EasyTuition
 EasyTuition - Find & Connect with Private Tutors Easily
EasyTuition is a simple web-based platform designed to help students find private tutors based on their subject, class level, and location. Teachers can create profiles, and students can browse, view detailed teacher information, and connect directly.
